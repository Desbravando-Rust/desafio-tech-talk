# Desafio Rust

Este projeto é desenvolvido em Rust. Siga os passos abaixo para instalar o Rust e rodar o projeto.

## Instalação do Rust

1. **Instale o Rust** utilizando o [rustup](https://rustup.rs/):

    ```sh
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
    ```

2. **Adicione o cargo ao PATH** (reinicie o terminal ou execute):

    ```sh
    source $HOME/.cargo/env
    ```

3. **Verifique a instalação**:

    ```sh
    rustc --version
    cargo --version
    ```

## Rodando o Projeto

1. **Clone o repositório**:

    ```sh
    git clone https://github.com/seu-usuario/desafio_rust.git
    cd desafio_rust
    ```

2. **Compile e execute o projeto**:

    ```sh
    cargo run
    ```
